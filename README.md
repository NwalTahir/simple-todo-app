# 🚀 Minimalist To-Do Application (React + Vite)

A modern, responsive, and lightweight task management web application designed to deliver an exceptional user experience with absolute zero bloat. Built using **React** for declarative, component-driven UI and powered by **Vite** for ultra-fast development and optimized production builds.

## ✨ Key Features & Technical Highlights
* **Optimized State Management:** Smooth, real-time interactive UI utilizing React's core hooks for instant task updates.
* **Blazing Fast Performance:** Leverages Vite’s Next-Generation frontend tooling for near-instantaneous Hot Module Replacement (HMR).
* **Clean & Scalable Architecture:** Structured with modular component design, keeping the codebase pristine, maintainable, and ready for future feature scalings.
* **Zero-Lag UI:** Lightweight styling and execution flow ensuring a seamless experience across all modern desktop and mobile browsers.

## 🛠️ Tech Stack
* **Frontend Library:** React.js
* **Build Tool:** Vite
* **Language:** JavaScript (JSX)

## 📦 Installation & Setup
To run this project locally, follow these steps:

1. Clone the repository:
  
   git clone [https://github.com/NwalTahir/simple-todo-app.git] (https://github.com/NwalTahir/simple-todo-app.git)

2.Navigate into the project directory:

cd simple-todo-app

3. Install dependencies:

npm install

4. Start the development server:

npm run dev -- --open
